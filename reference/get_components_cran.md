# Extract ggplot2 grammar components from a CRAN package

Extract ggplot2 grammar components from a CRAN package

## Usage

``` r
get_components_cran(pkg_name, cran_db = NULL)
```

## Arguments

- pkg_name:

  Package name

- cran_db:

  Optional CRAN package database

## Value

A data.frame with package, component, and function names
