# Extract ggplot2 grammar components from a GitHub repository

Extract ggplot2 grammar components from a GitHub repository

## Usage

``` r
get_components_github(pkg_name, repo_url)
```

## Arguments

- pkg_name:

  Package name

- repo_url:

  GitHub repository URL (optionally including /tree/branch/path)

## Value

A data.frame with package, component, and function names
